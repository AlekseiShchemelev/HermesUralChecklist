# Настройка авторизации

## Шаг 1: Создайте лист "Пользователи"

В вашей Google таблице создайте новый лист с названием **"Пользователи"**.

## Шаг 2: Добавьте заголовки

В первой строке добавьте заголовки колонок:
```
| ФИО | ПАРОЛЬ | РОЛЬ |
```

## Шаг 3: Добавьте пользователей

Заполните таблицу пользователями:

| ФИО | ПАРОЛЬ | РОЛЬ |
|-----|--------|------|
| Иванов И.И. | 123456 | admin |
| Петров П.П. | 789012 | user |
| Сидоров С.С. | 345678 | user |

**Роли:**
- `admin` - полный доступ
- `user` - обычный доступ (можно расширить функционал)

## Шаг 4: Обновите Google Script

1. Откройте ваш Google Apps Script проект
2. Замените код на новый из файла `GOOGLE_SCRIPT.js`
3. Сохраните и опубликуйте новую версию

## Шаг 5: Проверьте работу

1. Откройте приложение
2. Должна появиться форма входа
3. Введите ФИО и пароль из таблицы
4. После успешного входа отобразится основное приложение

## Безопасность

- Пароли хранятся в открытом виде (для простоты)
- Сессия действует 24 часа
- Для выхода нажмите кнопку "Выйти" в правом верхнем углу
- При обновлении страницы авторизация сохраняется

## Добавление новых пользователей

Просто добавьте новую строку в лист "Пользователи":
1. Откройте Google таблицу
2. Перейдите на лист "Пользователи"
3. Добавьте строку с ФИО, паролем и ролью
4. Новый пользователь может сразу войти
