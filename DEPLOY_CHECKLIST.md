# Чеклист деплоя в продакшен

## 1. Google Apps Script

### Обновление кода
- [ ] Открыть https://script.google.com
- [ ] Создать новый проект или открыть существующий
- [ ] Вставить код из `GOOGLE_SCRIPT.js`
- [ ] Сохранить (Ctrl+S)

### Развёртывание
- [ ] Нажать "Deploy" → "New deployment"
- [ ] Тип: Web app
- [ ] Execute as: Me
- [ ] Who has access: Anyone
- [ ] Скопировать URL

### Проверка CORS
- [ ] Открыть URL в браузере с `?action=get`
- [ ] Проверить, что возвращается JSON (не HTML)
- [ ] Проверить наличие заголовков CORS в DevTools

## 2. Фронтенд (GitHub Pages)

### Обновление URL
- [ ] В `js/config.js` заменить `appsScriptUrl` на новый URL

### Проверка версии Service Worker
- [ ] В `sw.js` увеличить `VERSION` (например, с '2' на '3')

### Загрузка файлов
- [ ] Все файлы из репозитория загружены в `main` ветку
- [ ] Проверить, что `manifest.json` и `sw.js` в корне

### Настройка GitHub Pages
- [ ] Settings → Pages → Source: Deploy from a branch
- [ ] Branch: main, folder: / (root)
- [ ] Подождать 2-5 минут

## 3. Проверка функциональности

### Ввод данных
- [ ] Открыть приложение
- [ ] Заполнить форму
- [ ] Нажать "Отправить"
- [ ] Проверить, что данные появились в Google Sheets

### Просмотр данных
- [ ] Переключиться на вкладку "Таблица"
- [ ] Проверить загрузку данных
- [ ] Проверить пагинацию (кнопка "Загрузить ещё")
- [ ] Проверить фильтрацию

### Редактирование
- [ ] Нажать "Редактировать" на записи
- [ ] Изменить данные
- [ ] Сохранить
- [ ] Проверить обновление в Google Sheets

### Отчёты
- [ ] Переключиться на вкладку "Отчёты"
- [ ] Проверить отображение графиков
- [ ] Проверить фильтр по датам

### PWA
- [ ] Проверить установку на телефон (Add to Home Screen)
- [ ] Проверить работу офлайн (включить Airplane mode)
- [ ] Проверить кэширование (второе открытие должно быть мгновенным)

## 4. Производительность

- [ ] Первая загрузка < 3 секунд
- [ ] Вторая загрузка (из кэша) < 1 секунды
- [ ] Рендер таблицы (100 записей) < 500 мс
- [ ] Открытие отчётов < 2 секунд

## 5. Безопасность

- [ ] В DevTools → Console нет ошибок CSP
- [ ] Нет ошибок CORS
- [ ] Данные экранируются (попробовать ввести `<script>alert(1)</script>`)

## 6. Откат

При проблемах:
1. В Google Script создать новый deployment с предыдущей версией
2. В GitHub откатить коммит
3. В `sw.js` увеличить VERSION ещё раз
