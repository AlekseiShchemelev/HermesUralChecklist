# Сменный чек-лист

Приложение для ведения сменных отчётов цеха. Данные сохраняются в Google Таблицу.

## 🚀 Быстрый старт

### 1. Настройка Google Таблицы

1. Создайте Google Таблицу
2. Добавьте заголовки в первую строку:
   - id, Дата, Смена, Цех, ФИО_мастера, Плазма_чел, Строжка_чел, Зачистка_чел, Авт_сварка_чел, Полотер_чел, Штамп_старый_чел, Италия_чел, Штамп_новый_чел, Отбортовка_чел, Кромко_чел, Котельщик_чел, Ручная_сварка_чел, Всего_чел, Плазма_листы, Строжка_сегменты, Авт_сварка_карты, Полотер_карты, Зачистка_карты, Штамп_старый, Штамп_италия, Штамп_новый, Комбинированные, Ремонтные, Отбортованные, Обрезанные, Упакованные, Пачек_в_пленку, Разгружено, Отгружено, Малая_печь, Большая_печь, Поломки, Дата_создания

### 2. Настройка Google Apps Script

1. Откройте `Расширения` → `Apps Script`
2. Вставьте код из файла `GOOGLE_SCRIPT.js`
3. Нажмите `Развернуть` → `Новое развертывание`
4. Тип: `Веб-приложение`
5. Выполнить как: `Я`
6. Кто имеет доступ: `Все`
7. Скопируйте URL и вставьте в `js/config.js` в поле `appsScriptUrl`

### 3. Публикация на GitHub Pages

1. Создайте репозиторий на GitHub
2. Загрузите все файлы проекта
3. В настройках репозитория включите GitHub Pages
4. Укажите ветку `main` и папку `/ (root)`
5. Дождитесь публикации (1-2 минуты)
6. Откройте сайт по ссылке `https://ваш-юзернейм.github.io/имя-репозитория/`

## ⚠️ Важно: Публикация на GitHub Pages

Приложение работает только в связке с Google Таблицей.

**Для локальной разработки:** Запустите через Live Server, но данные будут сохраняться только при публикации на GitHub Pages (из-за CORS ограничений).

**После публикации на GitHub Pages:**
- ✅ Данные сохраняются в Google Таблицу
- ✅ Редактирование и удаление работают
- ✅ Все функции доступны
