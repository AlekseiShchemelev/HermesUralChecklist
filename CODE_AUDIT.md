## После деплоя необходимо:

1. **Обновить Google Script** на сервере:
   - Открыть https://script.google.com
   - Обновить код из GOOGLE_SCRIPT.js
   - Опубликовать новую версию

2. **Создать лист "Пользователи"** в Google Sheets:
   - Колонки: ФИО | ПАРОЛЬ | РОЛЬ
   - Заполнить пользователями

3. **Очистить кэш браузера** (DevTools → Application → Clear site data)
